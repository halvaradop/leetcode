## Objetive

Transform the input array so that the first `k` elements contain only the unique values from the original array in their original relative order. The rest of the array can be ignored. Return `k`.

## Approach

This is a two-pointer problem â€” a common pattern when working with sorted arrays and in-place operations.

### Intuiton

Since the array is sorted, all duplicates are adjacent. Our goal is to overwrite duplicates by shifting unique values to the left side of the array, without using extra space.

We use two pointers:

- index â†’ Tracks the position to place the next unique value (initialized at 0).
- i â†’ Iterates through the array from left to right (starting at 1).

On each iteration, we compare `nums[i]` with `nums[index]`. If they differ, it means we found a new unique value, so we increment `index` and assign `nums[index] = nums[i]`.

### Code

```ts title="26. Remove duplicates from sorted array"
export const removeDuplicates = (nums: number[]): number => {
  let index = 0
  const n = nums.length
  for (let i = 1; i < n; i++) {
    if (nums[i] !== nums[index]) {
      nums[index + 1] = nums[i]
      index++
    }
  }
  return index + 1
}
```
